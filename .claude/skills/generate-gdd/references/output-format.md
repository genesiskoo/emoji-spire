# GDD 출력 포맷 참조

이 파일은 `generate-gdd` 스킬이 GDD.md를 생성할 때 사용하는 테이블 포맷 예시입니다.
실제 GDD.md를 생성할 때는 이 예시 행을 소스 코드에서 추출한 실제 데이터로 교체하세요.

---

## 카드 목록 포맷

```markdown
| 이름 | 비용 | 타입 | 효과 |
|------|------|------|------|
| 스트라이크 | 1 | 공격 | 적에게 6 피해 |
| 디펜드 | 1 | 방어 | 블록 5 획득 |
| 강타 | 2 | 공격 | 적에게 10 피해 |
| 화염구 | 2 | 공격 | 적 하나에게 12 피해 |
| 이중 공격 | 2 | 공격 | 적에게 4 피해를 2회 |
| 약점 노출 | 1 | 스킬 | 적에게 취약 2 부여 |
| 철벽 | 2 | 방어 | 블록 12 획득 |
| 분노 | 0 | 스킬 | 힘 +2 (이번 턴) |
| 약화 | 1 | 스킬 | 적에게 약화 2 부여 |
| 배리어 | 3 | 방어 | 블록 18 획득, 카드 1장 드로우 |
```

**포맷 규칙:**
- 타입은 한국어로 표기: `attack` → 공격, `defense` → 방어, `skill` → 스킬
- 비용은 숫자만 (단위 없음)
- 효과는 `description` 필드 그대로 사용

## 적 목록 포맷

```markdown
| 이름 | 이모지 | HP | 행동 패턴 |
|------|--------|-----|-----------|
| 슬라임 | 🟢 | 30 | 공격(6) / 방어(블록5) |
| 고블린 | 👺 | 25 | 공격(8) / 버프(힘+1) |
| 해골 | 💀 | 20 | 공격(6) / 이중타격(6×2) |
| 버섯맨 | 🍄 | 35 | 독포자(공격6+약화1) / 치유(HP+5) |
| 엘리트 | 👹 | 60 | 강타(14) / 방어(블록8) / 버프(힘+2) |
| 보스 | 🐉 | 120 | 강타(18) / 쓸기(전체8) / 방어(블록12) |
```

**포맷 규칙:**
- HP는 범위가 있으면 `min~max` 형식: 예) `25~35`
- 행동 패턴은 인텐트 이름과 수치를 `()`로 묶어 나열, `/`로 구분
- 일반/엘리트/보스를 구분하는 소제목 아래에 각 테이블 배치

## 이벤트 목록 포맷

```markdown
| 이름 | 선택지 | 효과 |
|------|--------|------|
| 치유의 샘 | 마시기 / 무시 | HP +15 / 없음 |
| 수상한 상자 | 열기 / 무시 | 랜덤 카드 획득 또는 HP -5 / 없음 |
| 행상인 | 거래 / 무시 | 골드 -30, 카드 1장 획득 / 없음 |
```

**포맷 규칙:**
- 선택지가 여러 개면 ` / `로 구분 (공백 포함)
- 효과도 선택지 순서에 맞춰 ` / `로 구분
- HP 변화는 `HP +N` 또는 `HP -N` 형식
- 골드 변화는 `골드 +N` 또는 `골드 -N` 형식
- 카드 획득은 `카드 획득`, 카드 제거는 `카드 제거`
- 아무 효과 없으면 `없음`

## 요약 통계 포맷

```markdown
| 항목 | 수치 |
|------|------|
| 총 카드 수 | 11장 |
| 평균 카드 비용 | 1.5 에너지 |
| 공격 / 방어 / 스킬 카드 | 5 / 3 / 3 |
| 총 적 수 | 6종 |
| 평균 적 HP | 48 |
| 일반 / 엘리트 / 보스 | 4 / 1 / 1 |
| 총 이벤트 수 | 3종 |
| 총 선택지 수 | 6개 |
```

**포맷 규칙:**
- 카드 수는 `N장`, 적/이벤트 수는 `N종`, 선택지는 `N개`
- 평균 비용은 소수점 1자리까지
- 평균 HP는 반올림하여 정수
